# Personal AI Assistant (JARVIS-inspired)

A full-stack personal AI assistant with a React frontend and Node.js backend.

## Project Structure

jarvise/
├── Jarvise/        # Frontend (React + Vite)
├── signinback/     # Backend (Node.js)

## Tech Stack
- Frontend: React, Vite
- Backend: Node.js
- Database: MySQL
- Voice & AI: pyttsx3, AI APIs

## Features
- User authentication (Sign up / Sign in)
- Voice interaction
- Command-based automation
